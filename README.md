# IBM-cognos-analytics

📘 KnowledgeHub Library Analytics

Library analytics project using IBM Cognos Analytics including reports, dashboards, explorations, and storytelling (Q1–Q6).

This project demonstrates library data analysis focusing on book circulation, overdue tracking, member behavior, and branch performance.

📊 Modules Covered
🔹 Q1 – Reports

Book Circulation Report (Genre & Branch-wise)

Overdue Books Summary

Fine Collection Analysis

Conditional Formatting for Delay Levels

🔹 Q2 – Member Insights

Overdue Members Report

Outstanding Fine Calculation

High Delay Identification (>30 days)

Branch-wise Member Analysis

🔹 Q3 – Circulation Trends

Books Issued by Branch

Monthly Circulation Trends

Format-wise Distribution (Digital / Audio / Print)

Interactive Chart with Filters

🔹 Q4 – Deep Exploration

Circulation Analysis by Genre

Member Borrowing Patterns

Overdue Behavior Analysis

Interactive Drill-down Insights

🔹 Q5 – Executive Dashboard

KPI Tiles (Total Members, Books Issued, Overdue Rate, Fine Collected)

Bar Chart (Genre vs Circulation)

Line Chart (Monthly Trends)

Pie Chart (Format Distribution)

Alert Table (High Overdue Cases)

🔹 Q6 – Data Storytelling

Library Overview

Branch Performance Analysis

Top Authors & Popular Books

Overdue & Fine Insights

Digital Usage Trends

Recommendations & Action Plan

🎯 Key Insights

Identified high overdue members and branches

Analyzed popular genres and formats

Tracked fine collection efficiency

Evaluated digital vs physical book usage

🚀 Action Plan

Improve overdue tracking system

Promote digital library usage

Reduce high delay penalties

Focus on high-demand genres

✅ Tools Used

IBM Cognos Analytics
